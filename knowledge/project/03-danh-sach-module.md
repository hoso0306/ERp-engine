# 03. Danh sách Module

Hệ thống ERP bao gồm các phân hệ (module) chính dưới đây:

| STT | Tên Module | Mã Module | Mô tả tóm tắt |
|-----|------------|-----------|---------------|
| 1 | Quản lý Mua hàng | `procurement` | Quản lý NCC, đơn mua hàng, nhận hàng |
| 2 | Quản lý Bán hàng | `sales` | Quản lý khách hàng, báo giá, đơn bán hàng, giao hàng |
| 3 | Quản lý Kho | `inventory` | Quản lý nhập/xuất/tồn kho, chuyển kho |
| 4 | Kế toán - Tài chính | `accounting` | Hóa đơn, sổ cái, thu chi, báo cáo tài chính |
| 5 | Quản trị Hệ thống | `system` | Phân quyền, cấu hình hệ thống, lịch sử hoạt động |
