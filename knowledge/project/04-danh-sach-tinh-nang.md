# 04. Danh sách tính năng

Chi tiết tính năng của từng Module:

## 1. Module Quản lý Mua hàng (`procurement`)
*   [ ] Quản lý thông tin Nhà cung cấp (Vendor)
*   [ ] Quản lý Đơn yêu cầu mua hàng (Purchase Requisition)
*   [ ] Quản lý Đơn mua hàng (Purchase Order)
*   [ ] Nhận hàng và kiểm tra chất lượng (Goods Receipt)

## 2. Module Quản lý Bán hàng (`sales`)
*   [ ] Quản lý thông tin Khách hàng (Customer)
*   [ ] Quản lý Báo giá (Quotation)
*   [ ] Quản lý Đơn bán hàng (Sales Order)
*   [ ] Hóa đơn bán hàng (Invoice)

## 3. Module Quản lý Kho (`inventory`)
*   [ ] Quản lý danh mục Sản phẩm, Vật tư
*   [ ] Nhập kho / Xuất kho
*   [ ] Điều chuyển kho nội bộ
*   [ ] Kiểm kê kho định kỳ
