# Dashboard

---

# Mục tiêu

Xây dựng Module Dashboard giúp chủ doanh nghiệp theo dõi nhanh tình hình hoạt động của doanh nghiệp.

Dashboard chỉ hiển thị dữ liệu tổng hợp, không thực hiện thêm, sửa hoặc xóa dữ liệu.

---

# Phạm vi

* Chỉ thực hiện Module Dashboard.
* Không sửa hoặc phát triển Module khác.
* Nếu thiếu nghiệp vụ, hỏi trước khi thực hiện.

# Yêu cầu

* Giao diện hiện đại, trực quan và dễ sử dụng.
* Responsive trên Desktop.
* Tuân thủ Tech Stack của dự án.
* Code sạch, dễ đọc, dễ bảo trì.
* Không thêm thư viện nếu chưa thực sự cần thiết.
* Chỉ đọc thêm tài liệu trong `knowledge/` khi thiếu thông tin.
